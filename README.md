 🚀 Projetos em Destaque:
- Dashboard de Consulta de Salas:
Aplicação web desenvolvida para centralizar informações acadêmicas de uma turma de aprendizagem profissional em uma única interface.
O projeto surgiu a partir de dois problemas enfrentados pelos alunos durante o curso:
- Consulta da escala anual:
A programação do curso e da empresa era disponibilizada em formato PDF e impresso, contendo todo o cronograma de aproximadamente um ano. A consulta diária dessas informações era pouco prática, exigindo que o aluno localizasse manualmente a data atual para verificar se deveria comparecer à empresa, ao curso técnico ou se havia feriado.
- Consulta da sala de aula:
As informações de salas eram disponibilizadas por meio de um painel que exibia dados de todas as turmas em rotação automática. Para localizar sua sala, o aluno precisava aguardar a exibição da página correspondente à sua turma, o que tornava a consulta lenta e pouco intuitiva.
  
Para solucionar esses problemas, foi desenvolvida uma aplicação que permite ao aluno visualizar rapidamente as informações relevantes para sua turma, reduzindo o tempo de consulta e melhorando a experiência de uso.

Principais funcionalidades:
- Consulta da sala de aula da turma para o dia corrente
- Visualização da programação diária (Empresa, Curso Técnico, Feriado ou Final de Semana)
- Exibição da escala semanal com navegação entre semanas
- Calendário letivo em visualização mensal e semanal
- Armazenamento local da turma selecionada
- Interface responsiva para dispositivos móveis e desktop

Tecnologias utilizadas:
- Frontend: Vue 3 (CDN), CSS puro com Glassmorphism  
- Backend: PHP 8 (single file)  
- Dados: JSON flat file (sem banco de dados)  
- API externa: localizada a chamada da api externa nesse ifram de consulta de salas disponibilizado para os alunos
- Hospedagem: IIS (Windows Server)  
- Ferramentas: Git, cURL

Esta é uma versão demonstrativa do projeto. Para preservar informações institucionais, os dados exibidos são simulados e a integração original foi substituída por mocks estáticos.
